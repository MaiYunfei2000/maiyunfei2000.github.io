Python 描述统计

发表于 2020-09-22 分类于 Mannual

注：

  1. 此系列文字面向已有统计基础的人，侧重 Python 实现而非统计知识的讲解。初学者请着重配合基础统计课程和教材食用。
  2. Python 版本默认为 3.6 。
  3. 此页面的程序会用到 Python 第三方库 NumPy 、 Pandas 和 StatsModels 。如果你还没有安装，请在命令行里通过 pip3 install numpy （或pandas、statsmodels）这些命令来安装（Windows 用户在“pip”后无需跟上“3”；若你不确定有无安装这些库，可使用pip3 list查看已安装的第三方库）。

1 知识回顾

1.1 集中性

平均数

用 ${x_1, x_2, …, x_i}$ 代表一个个数据， $N$ 为数据的个数，这些数据的平均数 (mean) 为：

$$\mu = \frac{x_1 + x_2 … + x_i}{N}$$

简化为：

$$\mu = \frac{\sum x_i}{N}$$

上面是总体平均数的公式。对于样本平均数，公式如下：

$$M = \frac{\sum x_i}{n}$$

也就是平均数和样本量的符号改变了一下。

注：研究报告中用$M$表示均值，而较多统计教科书用 $\bar{X}$ （读作“X bar”）。此系列文字默认使用前一种表示。

中数与百分等级分数

中数（或中位数、第二四分位数、第50百分位数）是将频数分布恰好分为两部分的数据。分布中恰有50%个数据的分数高于或低于中数。其符号表示为：

  • 中数或中位数 (median) ：$Mdn$
  • 第二四分位数 (2nd quartile) ：$Q_2$
  • 第50百分位数 (50th percentile) ：$P_{50}$

众数

一个频数分布中，众数 (mode) 是频数最大的数据。

注：如果你很无聊而且有点强迫症地关心它的符号表示，那么其中一种表示为 $M_o$ （下标为小写字母“O”），然而这个指标本来就用得不多，用到的时候也直接用“Mode”来表示就行了。

1.2 变异性

变异 (variance) 衡量一组数据的离散程度，数据越分散，它们的变异就越大。比如 ${1,3,5}$ 这组数据的变异性就大于 ${2,3,4}$ 这组数据。

注：在英文世界中方差与变异是一回事。

总体方差

$$\sigma^2 = \frac{\sum{x_i - M}}{N}$$

样本方差

$$s^2 = \frac{\sum{x_i - M}}{n - 1}$$

其中，可以用自由度 ($df$) 来表示 $n-1$ ：

$$s^2 = \frac{\sum{x_i - M}}{df}$$

标准差

方差的平方根： $\sigma$ 或 $s$ 。

注：至于为什么对样本变异性的计算要使用 $n-1$ 而非 $n$ ，参见各课程、教材或知乎等地方的帖子的介绍。

Q: 到底怎么判断是用总体还是样本的计算公式呢？

看你要用那批数据做什么。比如你就只是要看一批数据的离散程度，那这批数据就是一个总体；如果你要拿它去估计（推断）更大一批数据的情况，那这批数据就是一个样本。

像这篇知乎帖子里就讲得略详细，可以参考。总之，一句话，以实践需求为中心。

2 Python实现

例题：

对于数据 ${1, 5, 7, 3, 4, 4, 3, 2, 3} $，求它们的：

  • 平均数、中数、众数。
  • 方差、标准差（把数据当成样本数据）。
  • 方差、标准差（把数据当成总体数据）。

以下提供了几种实现方式，具体使用哪种方式，应结合你的实践情境来判断哪种最为方便。

2.1 内置函数

Python 标准库中提供了一个用于计算基础统计量的模块，叫作“statistics”。

PS：如官方所说：“此模块并不是诸如 NumPy，SciPy 等第三方库或者诸如 Minitab ，SAS，Matlab 等针对专业统计学家的专有全功能统计软件包的竟品。”

3.5555555555555554 3 3
1.740051084818425 3.0277777777777777
1.640535895581489 2.691358024691358

不过由于大多数情况下你不会仅仅需要算这些东西，所以这个库会比较鸡肋……

2.2 NumPy

NumPy 是一个提供了类似于 Matlab 的数组运算功能的第三方库。

  • numpy.array
  • numpy.mean

  • numpy.std

2.3 Pandas

Pandas 是易于使用的数据结构和数据分析的第三方库。你若是初入编程和计算领域，可以先将其粗暴地理解为 Python 上的表格工具。

你可以在编程时直接创建这个数据对象。然而实践情境中你需要打交道的数据量远大于例题使用的数据量，往往需要从外部文件中导入。对于这些情况，后面均有代码示例。

你需要准备好 csv 或 Excel 文件（xls 或 xlsx）。对于 csv ，你需要在任意编辑器中将以下文本保存为 .csv 文件（比如 data.csv）。

data
1
5
7
3
4
4
3
2
3

如果是 Excel ，请这样保存：

data

5
7
3

4

先在 Python 中创建数据，看看它长啥样：

   data

然后来看下统计量吧：

>>> data.describe()

mean   3.555556
std    1.740051
min    1.000000
25%    3.000000
50%    3.000000
75%    4.000000

如果你不仅仅是需要看一下结果，而是需要赋给变量，可以这样计算：

>>> data['data'].count()
9
>>> data['data'].mean()
3.5555555555555554
>>> data['data'].std()
1.7400510848184252
>>> data['data'].min()
1
>>> data['data'].quantile(0.25)
3.0
>>> data['data'].median()
3.0
>>> data['data'].quantile(0.75)
4.0
>>> data['data'].max()
7

关于 Pandas 的详细入门介绍，参见 Pandas 官方文档。
