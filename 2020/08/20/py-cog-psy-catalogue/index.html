发表于 2020-08-20 更新于 2021-01-11 分类于 Mannual
Handbook of Psychological and Cognitive Science in Python

挖坑说明

与其说是教程，倒不如说是自己的学习记录。如果这些内容是作为教程或参考手册，那么它要求读者有一定的心理或认知科学学习经验，以及一点 Python 编程的经验。也就是说这些资料尚不适合心理学与 Python 编程纯小白。

相当不定时地更新，每更新一篇内容，目录文字中的相应条目会变成超链接形式。行为部分按认知心理学主题来编排，其它部分按技术主题来编排。

绝大部分的坑在可见的未来都来不及去填（特别是测量和神经的部分；第一章“Python 及其生态”同样不会管，因为网上和市面上已有大把大把大把材料了）。此处仅仅是列出认知科学（特别是认知心理学、认知神经科学和计算认知科学）研究主要会遇到的内容和方法上的主题。如果填坑，填坑过程中可能会随具体内容编排而变更目录。
